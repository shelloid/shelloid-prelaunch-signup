About us

Shelloid Systems is a company formed by experienced developers with a passion for web development, cloud computing, and security. Our vision is to create enabling technologies for building robust and secure web application and services.

We bring our years of expertise across all layers of the web application and services stack - from the underlying protocols to web and application servers, to help you build dependable and secure web applications and services.

We offer the Shelloid Verify cloud service that does specification-based automated robustness testing for web APIs.

We believe in contributing back to the community. We develop the open source twin technology application server that integrates Node.js and Clojure and has an open, extensible architecture. The aim is to improve programmer productivity as well as software security and stability. The company offers enterprise services for the Shelloid application server users including cloud-based log intelligence, instant remote access of Shelloid-powered web applications, consultancy, training, and priority email support.

For consultancy, training, and enterprise support services for the Shelloid application server, please write to us at contact AT shelloid DOT com.
